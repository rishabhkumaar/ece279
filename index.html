OVERVIEW

EnviroSense — Industrial Air Quality Monitor

EnviroSense is a low-cost, high-reliability IoT system designed to continuously monitor air quality parameters in warehouses, manufacturing facilities, and industrial environments where hazardous gases, temperature extremes, or high humidity can pose safety risks.

PROBLEM STATEMENT

Industrial air quality degradation is responsible for a significant share of occupational health issues globally. Manual inspection is infrequent, costly, and slow to react to sudden changes in gas concentration. EnviroSense replaces periodic inspections with continuous, cloud-connected sensing.

PROJECT GOALS

  • Monitor MQ135 gas concentrations, temperature, and humidity in real-time
  • Transmit sensor readings to a Supabase PostgreSQL cloud backend
  • Provide a live web dashboard with visual alerts when thresholds are exceeded
  • Enable historical data analysis via logged records
  • Operate on low power — suitable for battery-backed deployment
HARDWARE

Hardware Architecture

The sensor node is built around the ESP8266 WiFi microcontroller, paired with industry-standard environmental sensors.

COMPONENTMODELSPECIFICATION
MicrocontrollerESP8266 (NodeMCU)80/160 MHz, 80 KB RAM, WiFi 802.11 b/g/n
Gas SensorMQ13510–300 ppm, detects NH3, NOx, CO2, smoke
Temp/HumidDHT22−40 to 80°C ±0.5°C, 0–100% RH ±2%
Power5V USB / 3.7V LiPo~80 mA active, deep-sleep capable
CloudSupabase (PostgreSQL)REST API, realtime, free tier
SOFTWARE

Software Stack

EnviroSense uses a lean, dependency-light stack to maximize performance and simplicity.

FIRMWARE (ESP8266)

// Core libraries
#include <ESP8266WiFi.h>
#include <ESP8266HTTPClient.h>
#include <DHT.h>

// Read sensors → format JSON → POST to Supabase
void loop() {
  float temp  = dht.readTemperature();
  float humid = dht.readHumidity();
  int   gas   = analogRead(A0);
  postToSupabase(gas, temp, humid);
  delay(3000);
}

USE CASES

Deployment Scenarios

EnviroSense is purpose-built for environments where air quality directly impacts worker safety and product integrity.

🏭
Chemical Warehouses

Detect volatile organic compounds and toxic gas leaks before they reach dangerous concentrations.

❄️
Cold Storage Facilities

Monitor temperature and humidity deviations that could spoil perishable goods.

🔋
Battery Storage Rooms

Detect hydrogen gas off-gassing from lead-acid or lithium battery banks.

🏢
Office Buildings

CO2 and VOC monitoring for HVAC optimization and occupant wellbeing.

ALERT LOGIC

Threshold & Alert System

The dashboard applies a three-tier color-coded alert system based on sensor readings.

SENSORSAFE ✓WARNING ⚠DANGER ✕
MQ135 (ppm)< 100100–150> 150
Temperature (°C)15–2828–40> 40
Humidity (%RH)30–6060–80> 80

When MQ135 exceeds 150 ppm, the dashboard activates a full-screen alert banner with a red pulsing glow effect on the gas card. Future versions will support email/SMS alerts via webhooks.

ROADMAP

Future Scope

EnviroSense is designed for extensibility. Planned enhancements include:

  • Multi-node support — Deploy multiple sensors across a facility with a unified map view
  • Push notifications — Integrate Twilio/SendGrid for real-time SMS and email alerts
  • ML anomaly detection — Detect irregular patterns before they become emergencies
  • Mobile app — React Native companion app with push alerts
  • OLED local display — Show current readings on a display attached to the sensor node
  • LoRaWAN support — Enable deployment in areas without WiFi coverage
